github 添加歌曲信息

在自己的文章上边添加一个小型音乐播放器，可以让其他人在看你文章的同时，听你喜欢的音乐。一段好的音乐，可以缓解心情，也可以让读者了解一下自己。是不是有点心动了呢？come on !

  • 命令安装 hexo-tag-aplayer
  • 使用方法

  • 参数详情

    • title : 歌曲名字（如：火星人来过）
    • author: 歌手名字（如：joker）
    • url: 歌曲存放地址（如：songs/joker.mp3）
    • picture_url: 可选。歌曲图片地址（如：imgs/joker.jpg）
    • narrow: 可选。 风格种类
    • autoplay: 可选。 自动播放音乐，通过移动浏览器不支持
    • width:xxx: 可选。 播放器宽度（默认：100%）
    • lrc:xxx: 可选。 歌词信息地址
