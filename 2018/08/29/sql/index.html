这几天搭建本地环境，数据库导入的时候，mysql数据库出现MySQL server has gone away错误，查了一番，找到了解决方法……

打开navicat的菜单中的工具tools，选择服务器监控server monitor，然后在左列选择数据库，右列则点选变量variable表单项，寻找参数max_allowed_packet，将其值改大。
